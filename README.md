Live2D
======

搭載 AZUSA 使用的 Live2D 整合介面

Live2D.exe为可嵌入Azusa程序

Live2D_Debug.exe为测试程序

Azusa-Live2D exe 接口

函数接口详情见Function/README.md

### 支持平台

WIN XP：32（未测） 64 （未测）

WIN7:32（未测） 64（已测）

WIN8(8.1):32（未测） 64（已测）
